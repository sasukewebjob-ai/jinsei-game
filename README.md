# 🎲 波乱万丈人生ゲーム

どん底と大逆転のジェットコースターライフ！
ブラウザだけで遊べる、2〜6人パス＆プレイ専用のなんちゃって人生すごろく。

## 遊び方

1. `index.html` をブラウザで開く（ダブルクリックでOK・インストール不要）
2. 人数（2〜6人）と名前を決めてスタート
3. 自分の番が来たらルーレット（1〜10）を回して進む
4. 全員ゴールしたら総資産を精算して、いちばんお金持ちが優勝！

## 特徴

- 全96マス・分岐2箇所（🎓大学/🏃就職、🏡安定/🎰ギャンブル）
- 職業14種・結婚・子供・マイホーム・保険・株・借金 全部入り
- カジノ（ハイ＆ロー／一点賭け8倍）・宝くじ（1等¥10,000,000）
- お宝カード12種（ターボチケット・サボり券・逆転の切り札…）
- 中断してもターンの頭から再開できる自動セーブ付き

## 開発メモ

- 素のHTML/CSS/JS（ビルド・外部ライブラリなし）
- 動作検証: `python tests/autoplay.py [人数]`（Playwrightで1ゲーム自動完走）
