# 🎲 波乱万丈人生ゲーム

どん底と大逆転のジェットコースターライフ！
ブラウザだけで遊べる、2〜6人パス＆プレイ専用のなんちゃって人生すごろく。

## 遊び方

1. `index.html` をブラウザで開く（ダブルクリックでOK・インストール不要）
2. 人数（2〜6人）と名前を決めてスタート
3. 自分の番が来たらルーレットを回して進む（**ホイールをドラッグで弾いてもOK！**）
4. 全員ゴールしたら総資産を精算して、いちばんお金持ちが優勝！

## 特徴

- **全144マス・全10章**（公園→キャンパス→青春の街→安定/ギャンブル分岐→波乱の40代→熟年→黄金ロード→夕焼けのフィナーレ）
- **職業17種は早い者勝ち（1人1職）**。職業ゾーンで止まったマスの職業に就く。全職に固有能力（医者=事故無効、プロゲーマー=カジノ勝ち1.5倍、宇宙飛行士=戻るマス無効…）
- **家14種がマップに実在**。買うと色が付いて**自分の名前の表札**が立つ。火事になると黒こげに…
- **約束手形**：お金が足りないと1枚¥1,000,000を自動発行（ゴール時¥1,500,000で返済／早期返済¥1,200,000）
- 結婚・子供で車のコマにピンが増える。決闘マス・選択式イベント・カジノ・宝くじ・株・保険ぜんぶ入り
- お祝いカットイン・コイン演出・紙吹雪・BGM・ドラムロール結果発表＋**称号授与**
- 中断してもターンの頭から再開できる自動セーブ付き

## 開発メモ

- 素のHTML/CSS/JS（ビルド・外部ライブラリなし）
- 動作検証: `python tests/autoplay.py [人数]`（Playwrightで1ゲーム自動完走）
