# 波乱万丈人生ゲーム（jinsei-game）

## 目的・ゴール
人生ゲーム風のすごろくボードゲームをブラウザだけで遊べる形で作る。
- 2〜6人の**パス＆プレイ専用**（1台のPCを回して遊ぶ。CPU・通信対戦なし）
- `index.html` をダブルクリックで起動できること（file:// で動作）
- 全員ゴール後に総資産を精算し、1位を決める

## 技術スタック
- 素のHTML/CSS/JS（ビルドなし・外部ライブラリなし）
- ES modules不使用（file://対応のため `<script>` タグ複数読み込み）
- 盤面はSVG描画、効果音はWebAudioで自前生成（音声ファイルなし）
- セーブはlocalStorage（ターン頭で自動保存→「続きから」で再開）

## やらないこと・制約
- オンライン対戦・通信機能
- スマホ最適化（PC優先。崩れない程度でOK）
- 外部ライブラリ・ビルドツール

## デザイン方向性
- カラフルでポップな王道すごろく盤面（曲がりくねった道＋分岐2箇所）
- テーマ：王道人生すごろく × **超波乱万丈**（どん底⇔大逆転） × **ギャンブル満載**
- 本家のパロディ（なんちゃって）。文言・職業・イベントは全てオリジナル

## ゲーム仕様（確定）
- ルーレット1〜10で進む。約96マス・分岐2箇所（進路：大学/就職、中盤：安定/ギャンブル）
- フルセット要素：職業（一般8種/エリート6種）・給料日（通過で給料）・結婚（止まるマス、全員から祝儀5万）・子供（ゴールで1人50万）・家（8種、ゴール時売却ルーレット×0.5〜×3）・保険（生命30万/火災50万）・株（株価変動、暴騰×2/暴落÷2イベント）・借金（不足時10万単位で自動借金、ゴール時1.5倍返済）・お宝カード12種（山札は各2枚）
- ギャンブル：カジノ（ハイ＆ロー2倍/一点賭け8倍）・宝くじ（1枚10万、1等1000万=1%）
- ゴールボーナス：1位200万〜6位10万。最終総資産で順位決定
- 初期所持金 ¥500,000

## ファイル構成
```
index.html        … 画面4つ（タイトル/設定/ゲーム/結果）＋オーバーレイ3つ
css/style.css
js/data-cards.js  … 職業/家/保険/お宝カード/定数
js/data-board.js  … 全96マス定義（座標は行スペックから自動計算）＋squareIcon()
js/player.js      … createPlayer/fmt/wait/estimateAssets
js/sound.js       … WebAudio効果音
js/roulette.js    … ルーレットオーバーレイ（Promiseで結果を返す）
js/board.js       … SVG盤面描画・コマ移動アニメ・カメラ追従/全体表示/ドラッグ
js/ui.js          … 汎用モーダル(Promise)/手番交代/サイドバー/ログ/トースト
js/casino.js      … ハイ＆ロー/一点賭け/宝くじ（賭け金ステッパー付き）
js/game.js        … ステートマシン本体（ターン進行/マス解決/精算/セーブ）
js/main.js        … 画面遷移・セットアップ・各ボタン配線
```
