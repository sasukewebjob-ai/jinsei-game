# 波乱万丈人生ゲーム（jinsei-game）

## 目的・ゴール
人生ゲーム風のすごろくボードゲームをブラウザだけで遊べる形で作る。
- 2〜6人の**パス＆プレイ専用**（1台のPCを回して遊ぶ。CPU・通信対戦なし）
- `index.html` をダブルクリックで起動できること（file:// で動作）
- 全員ゴール後に総資産を精算し、1位を決める

## 技術スタック
- 素のHTML/CSS/JS（ビルドなし・外部ライブラリなし）
- ES modules不使用（file://対応のため `<script>` タグ複数読み込み）
- 盤面はSVG描画、効果音はWebAudioで自前生成（音声ファイルなし）
- セーブはlocalStorage（ターン頭で自動保存→「続きから」で再開）

## やらないこと・制約
- オンライン対戦・通信機能
- スマホ最適化（PC優先。崩れない程度でOK）
- 外部ライブラリ・ビルドツール

## デザイン方向性
- カラフルでポップな王道すごろく盤面（曲がりくねった道＋分岐2箇所）
- テーマ：王道人生すごろく × **超波乱万丈**（どん底⇔大逆転） × **ギャンブル満載**
- 本家のパロディ（なんちゃって）。文言・職業・イベントは全てオリジナル

## ゲーム仕様（確定）
- ルーレット1〜8で進む。**全144マス**・分岐2箇所（進路：大学/就職、中盤：安定/ギャンブル）・全10章構成（幼少期→職業ゾーン→青春の街→安定orギャンブル→波乱の40代→熟年の階段→黄金のラストスパート→フィナーレ）
- **職業**：分岐後の職業ゾーン（各6マス）で「止まったマスの職業に就く」。**職業は早い者勝ちで1人1職（重複不可）**。先客がいたら就けない。ゾーン終端の就職フェア（止まるマス）で無職は必ず就職（空き職業から3択）。一般10種/エリート7種
- **職業固有能力**（全17職に1つずつ）：医者=事故無効／弁護士=詐欺半減／消防士=火事無効／公務員=リストラ無効／農家=台風無効／会社員=給料日着地3倍／大工=家2割引／プロゲーマー=カジノ勝ち1.5倍／パイロット=移動の出目+1／宇宙飛行士=戻るマス無効／教授=子供ボーナス2倍／IT=株1株おまけ／板前=お祝い徴収2倍／美容師=結婚+50万／パティシエ=結婚・出産+20万／芸人=半減後+50万／YouTuber=損失マスで+10万／宇宙飛行士=手形返済が割安(1枚100万)
- バランス：学費¥3,000,000、一般職給料20〜45万・エリート職60〜90万
- 給料日：通過で給料、止まったらダブル給料（会社員は3倍）
- **約束手形**：不足時に1枚¥1,000,000を自動発行。ゴール時1枚¥1,500,000で返済、手番中に¥1,200,000で早期返済可（資産パネル）。「手形半減の証文」カードあり
- 家：14種が**マップに実物表示**（売出中=グレー+「売」のぼり／購入で色付き+**プレイヤーカラーの表札に名前**／火事で黒こげ／ゴール売却で再販売）。家マス4箇所、大工2割引、売却ルーレット×0.5〜×3
- **決闘マス2箇所**：相手を指名して互いにルーレット、負けが賭け金（50万/100万）支払い
- **選択式イベント3箇所**（怪しい投資話／庭に謎の老人／豪華客船）：選択と運で結果が変わる
- ギャンブル：カジノ（2倍/8倍）・宝くじ（1等1000万）・プラチナ宝くじ（1等5000万）・人生最後の大勝負（±300万）・お祝い徴収（金婚式/古希）
- 演出：お祝いカットイン（結婚/出産/家購入/大金/1等/ゴール）・車が道に沿って回転・**結果発表で称号授与**（ギャンブルの帝王/手形王/不動産王など）
- ゴールボーナス：1位300万〜6位20万。最終総資産で順位決定。初期所持金 ¥500,000

```
js/sound.js       … WebAudio効果音（tick/drum含む）
js/bgm.js         … 16ステップシーケンサーBGM（title/game/result 3曲）
js/fx.js          … コイン飛散・紙吹雪（HTMLオーバーレイ）
js/roulette.js    … ルーレット（ボタン回し・セクター通過カタカタ音）
js/board.js       … SVG盤面（本家風：マス自体が道の台形マス帯/白縁道路/回転テキスト/ストップ看板/祝バッジ/車型コマ＋家族ピン/カメラ）
js/ui.js          … 汎用モーダル(Promise)/手番交代/サイドバー/ログ/トースト
js/casino.js      … ハイ＆ロー/一点賭け/宝くじ（賭け金ステッパー付き）
js/game.js        … ステートマシン本体（ターン進行/マス解決/精算/セーブ）
js/main.js        … 画面遷移・セットアップ・各ボタン配線
```
