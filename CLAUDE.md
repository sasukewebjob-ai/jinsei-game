# 波乱万丈人生ゲーム（jinsei-game）

## 目的・ゴール
人生ゲーム風のすごろくボードゲームをブラウザだけで遊べる形で作る。
- 2〜6人の**パス＆プレイ専用**（1台のPCを回して遊ぶ。CPU・通信対戦なし）
- `index.html` をダブルクリックで起動できること（file:// で動作）
- 全員ゴール後に総資産を精算し、1位を決める

## 技術スタック
- 素のHTML/CSS/JS（ビルドなし・外部ライブラリなし）
- ES modules不使用（file://対応のため `<script>` タグ複数読み込み）
- 盤面はSVG描画、効果音はWebAudioで自前生成（音声ファイルなし）
- セーブはlocalStorage（ターン頭で自動保存→「続きから」で再開）

## やらないこと・制約
- オンライン対戦・通信機能
- スマホ最適化（PC優先。崩れない程度でOK）
- 外部ライブラリ・ビルドツール

## デザイン方向性
- カラフルでポップな王道すごろく盤面（曲がりくねった道＋分岐2箇所）
- テーマ：王道人生すごろく × **超波乱万丈**（どん底⇔大逆転） × **ギャンブル満載**
- 本家のパロディ（なんちゃって）。文言・職業・イベントは全てオリジナル

## ゲーム仕様（確定）
- ルーレット1〜10で進む。**全140マス**・分岐2箇所（進路：大学/就職、中盤：安定/ギャンブル）・全10章構成（幼少期→職業ゾーン→青春の街→安定orギャンブル→波乱の40代→熟年の階段→黄金のラストスパート→フィナーレ）
- **職業**：分岐後の職業ゾーンで「止まったマスの職業に就く」（職持ちは転職選択可）。ゾーン終端の就職フェア（止まるマス）で無職は必ず就職。一般8種/エリート6種
- 給料日：通過で給料、**止まったらダブル給料**
- 結婚（止まるマス、全員から祝儀5万）・子供（ゴールで1人50万、最大4人）
- 家：**14種**（山小屋〜月の土地付きドーム6000万）、家マス4箇所、ゴール時売却ルーレット×0.5〜×3
- 保険が活きるマス：事故・入院5箇所（生命保険）／火事2＋住宅被害2箇所（火災保険）
- 株（暴騰×2/暴落÷2）・借金（自動借金、ゴール時1.5倍返済）・お宝カード12種×2枚
- ギャンブル：カジノ（ハイ＆ロー2倍/一点賭け8倍）・宝くじ（10万、1等1000万）・**プラチナ宝くじ（100万、1等5000万）**・**人生最後の大勝負（±300万）**・お祝い徴収マス（金婚式/古希）
- 終盤は大金が動く：退職金+500万、会社売却+1500万、ロマンス詐欺-200万、老後資金詐欺（半減）など
- マスに止まると**増減額がモーダルに大きく表示**される
- ゴールボーナス：1位300万〜6位20万。最終総資産で順位決定。初期所持金 ¥500,000

## ファイル構成
```
index.html        … 画面4つ（タイトル/設定/ゲーム/結果）＋オーバーレイ3つ
css/style.css
js/data-cards.js  … 職業/家/保険/お宝カード/定数
js/data-board.js  … 全96マス定義（曲線レイアウト自動計算）＋squareIcon()＋ROAD_CHAINS
js/scenery.js     … 風景レイヤー（エリア背景帯＋装飾60点超：観覧車/学校/ネオン街/御殿…）
js/player.js      … createPlayer/fmt/wait/estimateAssets
js/sound.js       … WebAudio効果音（tick/drum含む）
js/bgm.js         … 16ステップシーケンサーBGM（title/game/result 3曲）
js/fx.js          … コイン飛散・紙吹雪（HTMLオーバーレイ）
js/roulette.js    … ルーレット（ボタン回し＋ドラッグ弾き回し・セクター通過カタカタ音）
js/board.js       … SVG盤面（曲線道路/立体マス/車型コマ＋家族ピン/floatText/カメラ）
js/ui.js          … 汎用モーダル(Promise)/手番交代/サイドバー/ログ/トースト
js/casino.js      … ハイ＆ロー/一点賭け/宝くじ（賭け金ステッパー付き）
js/game.js        … ステートマシン本体（ターン進行/マス解決/精算/セーブ）
js/main.js        … 画面遷移・セットアップ・各ボタン配線
```
